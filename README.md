# 后端开发环境配置

## 一般配置

1. clone项目到本地

2. 安装Django相关环境

   ```
   pip install django
   
   pip install djangorestframework
   
   pip install django-filter
   
   pip install psycopg2
   
   pip install djangorestframework-stubs
   
   pip install django-cors-headers
   ```
